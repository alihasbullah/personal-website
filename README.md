# Personal Website

Personal website by Ali Hasbullah, built with Hugo and PaperMod theme.

## Tech Stack

- [Hugo](https://gohugo.io/) (extended) - Static site generator
- [PaperMod](https://github.com/adityatelange/hugo-PaperMod) - Hugo theme
- [Cloudflare Pages](https://pages.cloudflare.com/) - Hosting & deployment

## Languages

- 🇮🇩 Bahasa Indonesia (default)
- 🇬🇧 English

## Local Development

```bash
# Clone with submodules
git clone --recursive https://github.com/alihasbullah/personal-website.git
cd personal-website

# Run development server
hugo server

# Build for production
hugo
```
